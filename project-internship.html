Temperature, Humidity and Airborne Dust Detection System

The IoT system developed is designed to detect temperature, humidity, and airborne dust. The system's functionality is divided into two main parts as follows: 1. On the hardware side, devices and various sensors are installed, and then code is written to allow the system to transmit data via an MQTT broker, which serves as an intermediary for data transmission within a single topic. 2. On the software side, Node-RED is used to receive the data and display the results on a dashboard, as well as to log the data into a database.

Here are some features of the IoT system:

  • The system is capable of instantly measuring temperature, humidity, and airborne dust.
  • The results can be displayed on the dashboard.
