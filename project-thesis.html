CalcuRush Game (Thesis Project)

CalcuRush, an Endless Runner educational game using Unity to reinforce Grade 7 students’ understanding of real number operations (+, –, ×, ÷). Players solve math questions while avoiding obstacles, starting with 3 lives and earning 10 points per correct answer. The game features two difficulty levels, scoring, lives system, and menus including Play, How to Play, Math Tips, Settings, and Exit, with music and sound effects for engagement.

Here are some features of the game:

  • Interactive gameplay.
  • Real-time score tracking.
  • Challenging levels based on user performance.
